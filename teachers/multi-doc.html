Docs Tab Builder Guide

Overview

Upload multiple files and combine them into a single Google Doc, with each file placed in its own tab. Set a document title, name each tab, choose import options, then sign in and create.

Supported File Types

.txt — Plain text files. Content is inserted as-is.

.md — Markdown files. Choose to insert raw text or preserve Markdown formatting (headings, bold, italic, lists become styled text).

.docx — Word documents. Text and basic structure (headings, lists, bold/italic) are extracted. Complex formatting like tables or images may be simplified.

.pdf — PDF files. Text is extracted page by page. Scanned/image-only PDFs will appear empty. Tables come through as loose text.

Adding Tabs

Click Add Tab or drag files onto the page. You can select multiple files at once. Each file becomes a separate tab in the Google Doc.

Import Modes

Plain text — Inserts raw content with no formatting.

Preserve Markdown — Converts Markdown syntax (headings, bold, lists) into Google Docs styles.

Preserve formatting — For .docx files, maps Word styles to Google Docs formatting.

Extract text — Pulls text content only, discarding layout.

Tab Names & Reordering

Click any tab name to edit it. This becomes the tab title in the Google Doc. Drag cards up or down to reorder tabs.

Google Sign-In

Click Sign in with Google to authorize the app. You will be asked to grant permission to create Google Docs and files in your Drive. The app only creates new documents — it cannot read or modify existing files.

Creating the Doc

Once signed in with at least one tab added, click Create Google Doc. The first file becomes the default tab; additional files are added as new tabs. When done, the doc opens in a new browser tab.

From Google Doc

Click Add from Google Doc to open your Google Drive and select an existing document. If the document has multiple tabs, you will be asked to pick which tab to import.

Content is copied as faithfully as possible: headings, bold, italic, underline, and bullet/numbered lists are preserved. Tables are converted to tab-separated text. Images, drawings, and headers/footers are not copied.

You must be signed in with Google to use this feature.

Privacy

All file processing happens locally in your browser. File contents are only sent to Google when you click Create. No data is stored on any server besides your Google Drive.
