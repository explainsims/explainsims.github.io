How to Play Fun Jeopardy

  1. Teams take turns selecting categories and point values
  2. After a question is revealed, the first team to be selected answers
  3. Correct answers earn the point value shown
  4. Incorrect answers lose the point value shown
  5. The team with the most points at the end wins
  6. All answers should be phrased as a question (starting with "What is..." or "Who is...")

CSV Format Instructions:

To create your own questions using the template:

  1. Click the "Copy Template" button above
  2. Paste the contents (Ctrl+V or Cmd+V) into a spreadsheet
  3. In Google Sheets:
    • Select all pasted data
    • Go to Data menu - "Split text to columns"
    • Select "Comma" as the delimiter
  4. Edit your questions in the spreadsheet
  5. Make sure answers are in Jeopardy format (starting with "What is..." or "Who is...")
  6. Save/export your spreadsheet as CSV before importing
