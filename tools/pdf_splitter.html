2 Split Mode

After page

Pages after the last split point are bundled into the final file.

3 Naming

By default, files are named using the original filename followed by a space and sequential two-digit number: filename 01, filename 02, etc. Prefix and suffix are joined with " - ".

All files use default numbering.

Name each split individually. Leave blank for default numbering on that file.
