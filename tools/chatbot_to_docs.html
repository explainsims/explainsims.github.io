Options Guide

Citations

Remove citations strips inline reference markers that chatbots add when they search the web — e.g. [1], 【1†source】, [cite:…] — and removes trailing reference lists (Sources, References, etc.).

LaTeX

Convert to images turns any LaTeX maths output from a chatbot into rendered equation images via CodeCogs, so they paste correctly into Google Docs.

Size controls the image resolution. Match font size scales automatically with your chosen font size; the fixed options (Small / Medium / Large) give consistent sizing regardless of font.

Font

Choose the font family and size you want the text to be when pasted into Google Docs.

Scale headers adjusts heading sizes proportionally to your chosen font size (e.g. H1 scales from 24 pt at 12 pt body). When unchecked, headings use fixed default sizes.

Table Width

Sets the width of every table to exactly fill the text area of your document. Choose your paper size and set Left / Right margins.

Paper sizes are portrait orientation — widths are: A4 = 210 mm (8.27 in), A3 = 297 mm (11.69 in), US Letter / Legal = 8.5 in, US Tabloid = 11 in.

The default is A4 with 1-inch margins on each side, matching Google Docs' defaults. All columns within a table are made equal width.

Switch between in and cm — values convert automatically. Lock keeps both margins identical; uncheck to set them independently.
