PDF to Markdown Guide

What it does

Turns a PDF into Markdown. Markdown is far more token-efficient than a PDF when you feed it to an AI assistant later, and it is easy to edit and paste anywhere.

Fast (local) mode

Extracts the text layer from the PDF entirely in your browser — instant, free, and completely private. Use it for "digital" PDFs (exported from Word, Google Docs, LaTeX, etc.) that already have selectable text.

It cannot read scans, handwriting, graphs, or images, and it does not reconstruct complex tables. For those, use Smart mode.

Smart (AI) mode

Renders each page to an image and sends it to a vision model, which reads the page the way a person would. This handles scans, handwriting, graphs, and tables.

It ignores any low-quality OCR layer a scanner added and reads the page visually instead. Handwriting is transcribed exactly as written; unreadable parts are flagged as [illegible].

Provider & API key

Smart mode needs your own API key from Anthropic, OpenAI, or Google. Pick a provider, choose a model, and paste the key. The key is stored only in this browser and sent straight to the provider — it never touches our servers.

Larger models cost more but read difficult pages more accurately. Start with a mid-tier model and move up if results disappoint.

Page render quality

Higher quality sends a sharper image to the model — better for fine handwriting or dense tables, but uses more tokens (higher cost) and is slower. Standard is fine for clean typed scans.

Figures, graphs & tables

In Smart mode, tables are transcribed as Markdown tables where possible. Photos, graphs, and diagrams are cropped out of the page, saved as images, and referenced in the Markdown at the right spot with a short caption.

Downloads

Download .md gives a single Markdown file. If figures were extracted, use Download ZIP — it bundles document.md with a figures/ folder so the image links resolve.

Privacy & cost

Local mode never leaves your device. Smart mode sends page images directly to your chosen AI provider under your own account — you are billed by them. Big PDFs mean many pages and higher cost, so check the page count first.
